# MongoDB
My custom API built. This database is used for the Food Truck Finder app for [iOS](https://github.com/Noel-F/Android) and [Android](https://github.com/Noel-F/Android).



### Built With
* [MongoDB](https://www.mongodb.com/)
* [ExpressJS](https://expressjs.com/)
* [Es6](https://developer.mozilla.org/en-US/docs/Web/JavaScript/New_in_JavaScript/ECMAScript_2015_support_in_Mozilla)
* [NodeJS](https://nodejs.org/en/)
* [MoongooseJS](http://mongoosejs.com/index.html)
* [Body-Parser](https://github.com/expressjs/body-parser)
* [Nodemon](https://nodemon.io/)
* [BabelJS](https://babeljs.io/)
